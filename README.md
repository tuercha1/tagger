# AI 绘图数据集标签管理器

这是一个比较方便快捷的 AI 绘图数据集标签管理器。

它可以帮助你在浏览器里管理图片数据集、批量生成标签、编辑单张图片标签，并维护常用标签翻译词库。适合整理 Stable Diffusion、LoRA 训练等场景下的图片和标签文件。

我不太会写文档，所以我决定摆烂不写，总之直接启动大概能用。

## 功能

- 浏览本地图片数据集文件夹
- 扫描图片并显示已有标签状态
- 使用 WD/EVA02 标签模型进行批量打标
- 单张图片重新识别和手动编辑标签
- 支持触发词、标签前缀、阈值等打标参数
- 标签翻译和本地翻译词库管理
- 翻译词库导入、导出

## 使用方式

需要 Python 3.11。

双击 `run.bat` 启动。脚本会自动创建 `.venv`、安装依赖，并打开：

```text
http://127.0.0.1:7860
```

首次启动后，在界面中选择模型目录即可使用。

## 配置

公开仓库只提供 `config.example.json` 示例配置。

本地使用时可以复制一份并命名为 `config.json`，也可以直接在界面里设置模型目录、翻译 API 等信息。

请不要提交自己的 `config.json`、API 密钥、数据库、日志或本地路径。
